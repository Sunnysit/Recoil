## 创建 React App

因为 Recoil 是用于 React 的一个状态管理库， 所以你需要先安装 React 才可以使用 Recoil。最简单和推荐的方式是使用[Create React App](https://github.com/facebook/create-react-app#creating-an-app)来创建 React App。

```shell
npx create-react-app my-app
```

> [`npx`](https://medium.com/@maybekatz/introducing-npx-an-npm-package-runner-55f7d4bd282b)是 npm 5.2 及更高版本随附的打包运行程序工具，请参阅[npm 较旧版本的说明](https://gist.github.com/gaearon/4064d3c23a77c74a3614c498a8bb1c5f)。

如果想了解更多安装 React App 的方式，请阅读[官方文档](https://github.com/facebook/create-react-app#creating-an-app)。

## 安装

Recoil 已发布在<a href="https://www.npmjs.com/get-npm" target="_blank">npm</a>。如果想安装最新的稳定版本，请运行以下命令行：

```shell
npm install recoil
```

如果你使用的是 <a href="https://classic.yarnpkg.com/en/docs/install/" target="_blank">yarn</a>:

```shell
yarn add recoil
```

## RecoilRoot

若组件需要使用 Recoil 的 state，首先需要把`RecoilRoot`组件放在父树的某个位置，建议放在根组件里：

```jsx
import React from 'react';
import {
  RecoilRoot,
  atom,
  selector,
  useRecoilState,
  useRecoilValue,
} from 'recoil';

function App() {
  return (
    <RecoilRoot>
      <CharacterCounter />
    </RecoilRoot>
  );
}
```

我们将会在接下来的章节创建`CharacterCounter`组件。

## Atom

一个**atom**代表一小块**状态**。Atoms 可以被任意组件读取或写入。读取 atom 值的组件隐式地**订阅**了该 atom，因此任何 atom 更新都将导致所有订阅了该 atom 的组件重新渲染：

```javascript
const textState = atom({
  key: 'textState', // unique ID (with respect to other atoms/selectors)
  default: '', // default value (aka initial value)
});
```

组件需要读取和写入 atom 时应该使用`useRecoilState（）`，如下所示：

```jsx
function CharacterCounter() {
  return (
    <div>
      <TextInput />
      <CharacterCount />
    </div>
  );
}

function TextInput() {
  const [text, setText] = useRecoilState(textState);

  const onChange = (event) => {
    setText(event.target.value);
  };

  return (
    <div>
      <input type="text" value={text} onChange={onChange} />
      <br />
      Echo: {text}
    </div>
  );
}
```

## Selector

一个**selector**代表一部分**派生的状态**。派生状态是状态的“转换”。 您可以将派生状态视为将状态传递给以某种方式修改给定状态的纯函数的输出：

```jsx
const charCountState = selector({
  key: 'charCountState', // unique ID (with respect to other atoms/selectors)
  get: ({get}) => {
    const text = get(textState);

    return text.length;
  },
});
```

我们可以用 `useRecoilValue()` hook 来读取`charCountState`的值：

```jsx
function CharacterCount() {
  const count = useRecoilValue(charCountState);

  return <>Character Count: {count}</>;
}
```
